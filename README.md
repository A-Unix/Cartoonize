## Installation instructions
 ### Clone the repo
  ```bash
  git clone https://github.com/A-Unix/Cartoonize.git
  ```
 ### Move to cloned folder
 ```bash
 cd Cartoonize
 ```
 ### ### Get permissions
  ```bash
  chmod +x setup.sh main.py
  ```
 ### Install dependencies
  ```bash
  ./setup.sh
  ```
 ### Run the main file
  ```bash
  python3 main.py
  ```
Enjoy :)
